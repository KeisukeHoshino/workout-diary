メニューを作成

複数テンプレート時の選択ルール

  • 「テンプレート適用」押下でメニューテンプレート選択シートを開く。
  • テンプレートは `MenuTemplate.sortOrder` 昇順で表示する。
  • 各行にはメニュー名、登録種目数、主な種目名を表示する。
  • ユーザーが1件を選択すると `applyMenuTemplateToDate(date, menuTemplateId)` を実行する。
  • テンプレートが0件の場合は空状態と「メニューを作成」導線を表示する。

2. 画面操作設計

操作イベント結果
日付移動`changeDate(date)`URL query を更新し対象日の詳細を再取得。
体重入力`upsertBodyWeight(date, value)`400ms デバウンスで日付ごとに保存。
種目追加`addExerciseToDate(date, exerciseId)`既存カードがあればセット追加、なければカードと空セットを作成。
テンプレート適用開始`openMenuTemplateSheet()`メニューテンプレート選択シートを開く。複数テンプレートがある場合は一覧から1件を選ぶ。
メニューテンプレート選択`applyMenuTemplateToDate(date, menuTemplateId)`選択したテンプレート内の種目を表示順で展開する。既存カードがあれば空セットを追加し、なければ種目カードと空の1セット目を作成。
セット追加`addSet(workoutExerciseId)`前セットの重量・回数をコピー。
セット編集`updateSet(setId, patch)`対象行だけ保存状態を表示。

バリデーションとエラー表示

入力のたびに検証し、不正値は IndexedDB へ保存しない。エラーは対象入力欄の直下に表示し、値が正常になった時点で解除して自動保存を再開する。

対象エラー条件表示メッセージタイミングと動作
日付URL query またはカレンダー入力が実在する `YYYY-MM-DD` 形式ではない。日付を正しく入力してください日付決定時に検証する。対象日の取得と URL 更新を行わず、当日表示を維持する。
体重空欄以外で数値に変換できない。体重は数値で入力してください入力時に表示する。自動保存を開始しない。
体重0 未満または 999.9 を超える。体重は0〜999.9kgで入力してください入力時に表示する。自動保存を開始しない。
体重小数第2位以下が入力されている。体重は小数第1位までで入力してください入力時に表示する。値を丸めず、ユーザーの修正を待つ。
セット重量空欄以外で数値に変換できない。重量は数値で入力してください入力時に対象セット行へ表示する。対象行の自動保存を開始しない。
セット重量0 未満または 999.9 を超える。重量は0〜999.9kgで入力してください入力時に対象セット行へ表示する。対象行の自動保存を開始しない。
セット重量小数第2位以下が入力されている。重量は小数第1位までで入力してください入力時に対象セット行へ表示する。値を丸めず、ユーザーの修正を待つ。
回数空欄以外で整数に変換できない。回数は整数で入力してください入力時に対象セット行へ表示する。対象行の自動保存を開始しない。
回数1 未満または 999 を超える。回数は1〜999回で入力してください入力時に対象セット行へ表示する。対象行の自動保存を開始しない。
  • 体重、重量、回数は空欄を許容し、空欄へ戻した場合はエラーを解除して `null` として保存する。
  • 重量と回数は片方だけの入力も許容する。
  • 保存処理自体が失敗した場合はバリデーションエラーと区別し、「保存できませんでした。再試行してください」を対象欄または対象セット行へ表示する。

4. 重要コード

export async function addExerciseToDate(date: LocalDateString, exerciseId: UUID) {
  await db.transaction('rw', db.workoutDays, db.workoutExercises, db.workoutSets, async () => {
    const day = await ensureWorkoutDay(date);
    const existing = await db.workoutExercises.where({ workoutDayId: day.id, exerciseId }).first();

    if (existing) {
      await addSet(existing.id);
      return;
    }

    const workoutExercise = createWorkoutExercise({
      workoutDayId: day.id,
      exerciseId,
      sortOrder: await nextWorkoutExerciseSortOrder(day.id),
    });

    await db.workoutExercises.add(workoutExercise);
    await db.workoutSets.add(createEmptyWorkoutSet(workoutExercise.id, 1));
  });

  return getWorkoutByDate(date);
}
