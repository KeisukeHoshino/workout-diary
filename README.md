# 筋トレ日記

スマホで素早く筋トレと体重を記録するためのオフライン対応 Web/PWA アプリです。

## 使い方

`index.html` をブラウザで開くか、ローカルサーバーで配信してください。

```sh
python3 -m http.server 5173
```

その後、`http://localhost:5173` を開きます。

## 主な機能

- 日付ごとの体重、種目、セット重量、回数の記録
- IndexedDB による端末内保存
- プリセット種目からマイ種目への追加
- マイ種目の作成、非表示、復元
- メニューテンプレート作成と記録への一括追加
- 最大重量推移と体重推移のグラフ
- 日別履歴の確認と削除
- PWA manifest と Service Worker
